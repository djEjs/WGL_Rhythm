간단한 메트로놈 + Mp3 로드 후 BPM 싱크 테스트용 프로젝트 입니다.
- 메트로놈 BPM 조정은 60 -> 300 까지 가능합니다.
- 메트로놈의 비트 수는 1 -> 8 까지 가능합니다.
- 비트의 버튼은 체크 시 강, 체크 해제 시 약 비트가 재생됩니다.
- MP3 로드 시 파일 이름이 버튼 옆에 표시됩니다.

상단 버튼 4종
- TAB   : 메트로놈 BPM 수동 체크
- START : 메트로놈 시작
- PAUSE : 메트로놈 일시 정지
- STOP  : 메트로놈 정지


하단 버튼 4종
- LOAD  : 로컬 MP3 로드
- START : 로드한 MP3 재생
- PAUSE : 로드한 MP3 일시 정지
- STOP  : 로드한 MP3 정지

- MP3로드 및 스펙트럼 싱크 작업
